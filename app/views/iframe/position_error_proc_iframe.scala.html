ขั้นตอนการทดลอง Position Error

  1. ระบบจะทำการนับถอยหลังจาก 3 ถึง 0 เพื่อให้ท่านเตรียมตัวก่อนที่โจทย์จะปรากฏระบบจะแสดง
  2. ระบบจะแสดงโจทย์ ซึ่งเป็นตัวอักษรหรือตัวเลข มีจำนวนตั้งแต่ 3 - 12 ตัว โดยจะแสดงขึ้นมาทีละ 1 ตัว ให้ท่านจำตัวอักษรหรือตัวเลขเหล่านี้ไว้
  3. เมื่อโจทย์หายไป ระบบจะแสดงตัวอักษรโจทย์ขึ้นมาอีกครั้ง แต่เรียงลำดับสลับตำแหน่งกัน
  4. ให้ท่านจัดเรียงตัวอักษรเหล่านั้น ตามลำดับที่ท่านจำไว้ โดยท่านสามารถ Drag and Drop ตัวอักษรจากแถวบน มายังแถวล่างเพื่อจัดเรียงลำดับ
  5. เมื่อท่านจัดเรียงลำดับเสร็จแล้ว ให้ท่านกดปุ่มยืนยันเพื่อส่งคำตอบและไปยังข้อถัดไป

    6. ผลการทดลองที่ได้ไม่ได้ระบุถึงความสามารถหรือIQแต่แสดงให้เห็นว่าการประมวลสารสนเทศของมนุษย์สามารถได้รับอิทธิพลจากสิ่งเร้าหลายรูปแบบ
